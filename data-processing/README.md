INHABIT

This folder contains following files: 

NoaaSchemaDescription: This file contains description about format of NOAA data set. Original dataset for this can be found at
https://www.ncdc.noaa.gov/crn/qcdatasets.html

OpenAQJsonDescription: This file contains description about OpenAQ data. The link to this dataset can be found here:
 https://registry.opendata.aws/openaq/

spark_processing.py: Python code to process data sets

sparksubmit.sh: spark submit command to run spark_processing.py script
